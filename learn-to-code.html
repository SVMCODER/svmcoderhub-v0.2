What is HTML?

It stands for HYPERTEXT MARKUP LANGUAGE. It is used for creating webpages containing both text and graphics.

It was previously known as WYSIWYG: what you see is what you get.

It's code looks like this

< !DOCTYPEhtml >
< html >
< head >
Head content
< /head >
< body >
Body here
< /body >
< /html >


Things required to code a webpage using HTML.

1. A Text Editor [To code the website]
2. A Browser [To view the website]


What are headings?

Headings are used to give importance to a sentence. From h1 to h6.
H1 is used when the heading is so much important
H6 is used when the heading is almost not important at all.


Usage
< h1 > Hello < /h1 >
Output

Hello


What is an elemnet?

An elemnet consists of two types.

ON and OFF

In ON, we add our attributes and OFF is empty.

< h6 class="example"> Word < /h6 >


Output

Word
