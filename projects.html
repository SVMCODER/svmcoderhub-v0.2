Image in a webpage

In this post, I am gonna teach you how to insert an image inside a webpage. Image in a website can only be visible to some of the browsers and nowdays, It is almost available in every browser like Google Chrome, Miscrosoft edge, Safari, Bing, etc...

Image tag consists only ON tag.

< img >

We need to add the destination of the image file using src (source) attribute and an alternative name using alt (alternative) attribute.

< img src="image.png" alt="an image" >
